智能影评与推荐系统

项目简介
一个基于智能分析的电影评论与推荐系统，能够分析用户评论情感并提供个性化电影推荐。

主要功能
- 🎬 电影信息库管理
- 💬 用户评论与评分系统
- 🧠 评论情感分析（正面/负面）
- 🔍 智能电影推荐（协同过滤 + 内容过滤）
- 📊 热门电影榜单

技术栈
- 后端： [Spring Boot, Django, Node.js 等]
- 前端： [Vue, React, HTML/CSS/JS 等]
- 数据库： [MySQL, MongoDB 等]
- 情感分析： [SnowNLP]
